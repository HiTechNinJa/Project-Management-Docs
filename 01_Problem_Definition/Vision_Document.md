# 项目愿景文档
## 1. 待解决的核心问题
- 解决“隐私保护”与“精准监控”的矛盾（雷达无画面，仅输出坐标数据）；
- 解决监控设备“误报率高”问题（硬件级区域过滤+软件级阈值判定）；
- 解决“实时性”与“低功耗”的平衡（动态调整上报频率）；
- 提供直观的可视化操作界面（替代传统雷达的纯数据输出）；
- 支持远程配置与历史数据复盘（满足用户追溯与管理需求）。

## 2. 目标用户群体
- 个人用户：家庭安防（夜间守卫、宠物/老人活动监控）、个人工作室空间管理；
- 小型商业用户：便利店、办公室、仓库的区域监控（无隐私泄露风险）；
- 物联网开发者：需要雷达数据可视化、远程控制的二次开发场景。

## 3. 项目核心价值
- 隐私优先：仅采集坐标、速度等非敏感数据，无画面泄露风险；
- 智能高效：动态频率控制兼顾实时性与低功耗，区域过滤降低误报；
- 易用性强：Web 端可视化操作，支持拖拽划区、一键模式切换；
- 可扩展性：开放 API 接口，支持二次开发与设备接入扩展。
