# WatchYourBehind - 智能雷达可视化系统

本项目为 **2025 软件工程实训** 项目文档仓库。

## 1. 项目简介
**WatchYourBehind** 是一款基于 **HLK-LD.2450** 毫米波雷达的智能可视化监测系统。本项目旨在通过对雷达数据的采集、处理与可视化展示，实现精准的目标监测与预警功能，可应用于智能家居、安全监测等场景。
